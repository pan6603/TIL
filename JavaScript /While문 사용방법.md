## While문 무엇인가
> while문 조건이 true 일 경우 계속 반복하여 순회 하는 것을 말한다. 만약 false 일 경우 반복하지 않고 종료된다.

### 사용 방법

1. true 일 경우 실행하기 
```
while (조건) { // true 일 경우 실행
  // 실행할 코드
}
```
+ 조건이 true일 때 코드 블록을 실행합니다.
+ 조건이 false가 되면 반복을 멈춥니다.

2. 1: 1부터 5까지 출력
```
let i = 1;

while (i <= 5) {
  console.log(i);
  i++; // 반복을 멈추기 위해 i 값을 증가시킴
}

```
+ true 일 경우 i++; 증가시킴
+ i 값이 5 일 경우 false가 되므로 실행 종료
